# v-jsonschema-form

Данный пакет является портированной версией библиотеки react-jsonschema-form (https://github.com/rjsf-team/react-jsonschema-form).

Пакет включает в себя поддержку следующих полей и виджетов из оригинальной библиотеки

Поля:

- ArrayField
- BooleanField
- DescriptionField
- MultiSchemaField
- NumberField
- ObjectField
- SchemaField
- StringField
- TitleField

Виджеты:

- CheckboxesWidget
- CheckboxWidget
- PasswordWidget
- RadioWidget
- RangeWidget
- SelectWidget
- TextareaWidget
- TextWidget
- UpDownWidget
- URLWidget
